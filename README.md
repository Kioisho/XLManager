# XLManager

Administrador de CFDI, descargas del SAT y reportes confiables de Excel para Windows.

## Primera instalación

Descarga el MSI más reciente desde [Releases](https://github.com/Kioisho/XLManager/releases/latest) y ejecútalo. El instalador y la aplicación están firmados digitalmente.

## Actualizaciones

XLManager puede buscar e instalar nuevas versiones desde **Configuración > Actualizaciones**. Antes de instalar, valida la integridad SHA-256 y exige la firma autorizada de XLManager.

## Desarrollo

Usa GitHub Desktop para sincronizar, revisar y confirmar cambios, y abre `XLManager.slnx` con Visual Studio Community para desarrollar. El repositorio contiene la aplicación, las pruebas y el instalador WiX.

La guía para preparar Git y publicar una actualización está en [docs/PUBLICAR.md](docs/PUBLICAR.md).
